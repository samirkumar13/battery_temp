# 🔋 Battery Temp

![Android](https://img.shields.io/badge/Platform-Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)
![Kotlin](https://img.shields.io/badge/Language-Kotlin-7F52FF?style=for-the-badge&logo=kotlin&logoColor=white)

A beautifully simple, lightweight, and **100% ad-free** Android application to monitor your device's battery temperature in real-time.

---

## 🎯 Why This App?

Have you ever looked for a simple app just to check your battery temperature, only to find the market flooded with apps that are:
- 📉 Cluttered with unnecessary "boost" or "cleaner" features.
- 🛑 Loaded with intrusive, annoying ads.
- 🔋 Ironically, draining your battery in the background.

**Battery Temp** was created with a single purpose: to give you accurate battery temperature readings in a clean, elegant, and distraction-free interface. No ads. No bloat. Just the data you need.

## ✨ Features

- **Real-Time Monitoring**: Get accurate battery temperature readings instantly.
- **Clean UI**: A modern, aesthetically pleasing interface built with Jetpack Compose.
- **Zero Ads**: Completely ad-free and privacy-respecting.
- **Lightweight**: Minimal impact on your system resources and battery life.

## 🚀 How to Install & Use

### Option 1: Download the APK (Easiest)
You can directly download the pre-built APK and install it on your Android device.
1. Go to the [Releases](https://github.com/samirkumar13/battery_temp/releases) page of this repository. *(Or look in the `app/release/` folder in the source code!)*
2. Download the latest `.apk` file.
3. Open the file on your Android device to install it (you may need to allow "Install from unknown sources" in your settings).

### Option 2: Build from Source
If you are a developer and want to build the app yourself:
1. Clone the repository:
   ```bash
   git clone https://github.com/samirkumar13/battery_temp.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync and build the project.
4. Connect your device and click **Run** (▶️).
